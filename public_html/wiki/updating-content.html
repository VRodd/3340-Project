How to Update Site Content

This guide is for site administrators who need to add, remove, or modify products and media.

Adding a New Product

Open js/main.js and locate the products array. Add a new object with the fields: id, name, category, price, option, and img.

Changing Product Info

Find the product in the products array by its id and edit the details. Save the file — the shop page and search will update automatically.

Updating Images

Place new images in the images/ directory. Reference them by filename in the product object. Use copyright-free images only.

Changing Themes

Theme CSS variables are defined in css/styles.css under :root, [data-theme="dark"], and [data-theme="midnight"].
